# Orkait Website

A Next.js 16 project with React 19, Tailwind CSS v4, and shadcn/ui.

## Installation

### Prerequisites

- Node.js 18+
- npm or bun

### Using npm

```bash
# Clone the repository
git clone <repo-url>
cd orkait-3.0

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Using bun

```bash
# Clone the repository
git clone <repo-url>
cd orkait-3.0

# Install dependencies
bun install

# Start the development server
bun dev
```

## Available Scripts

| Command          | npm             | bun             |
| ---------------- | --------------- | --------------- |
| Dev server       | `npm run dev`   | `bun dev`       |
| Production build | `npm run build` | `bun run build` |
| Start server     | `npm run start` | `bun start`     |
| Lint             | `npm run lint`  | `bun lint`      |

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.
